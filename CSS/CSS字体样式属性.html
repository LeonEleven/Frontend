企业简介

公司由来

2006年，几个热血青年怀揣着对教育事业的拳拳之心，一起探讨中国教育的发展方向，同年5月8日传智播客正式成立，改变中国IT教育的星星之火由此点燃。历经13年风雨，传智播客从当年那不足10人的小团队，已发展成为现在拥有2000多名员工的教育集团；从成立最初的单一Java学科，到现在已包括JavaEE、Python+人工智能、前端与移动开发、UI/UE设计、大数据、Go语言与区块链等14门学科培训；从屈指可数的几间教室，发展成为分布于北京、上海、广州、深圳、武汉、郑州、西安、长沙、济南、重庆、南京、杭州、成都、石家庄、合肥、太原、厦门、沈阳等18所城市直营中心的规模。目前旗下也已拥有了“黑马程序员”、 “博学谷”、“传智专修学院”、“酷丁鱼”、“传智汇”、“院校邦”等子品牌。

我们的标识

传智播客的Logo 由图案和文字两部分组成。 从直观上看，图案由靶子和靶心构成，文字为公司名称和网址。图案代表公司的发展目标，即“探索教育本源，开辟教育新生态”的终极愿景。蓝色的靶子，体现了浓浓的科技感；红色的靶心，彰显了三层含义，分别是：用良心做教育、齐心协力、心想事成。传智播客汇聚了一帮传递智慧、播撒知识的侠客，我们相信只要“传智人”永远本着良心做教育，齐心协力，最终一定可以心想事成。

我们的使命

让万千学子在求知的道路上少走弯路；帮助更多迷茫的年轻人用知识和技能改变命运，是传智播客创立的初心，也是传智播客自创始以来始终坚守的使命与责任。

我们的愿景

探索教育之路，长途漫漫。传智播客希望可以通过自己的努力，寻找出一条更符合人类自然成长规律的教育之路，建立起一个新的教育生态环境，让中国的家长和孩子们在现有的教育体系之外，再多一些选择的机会。因此，“探索教育本源，开辟教育新生态”便成为了所有传智人为之奋斗的终极愿景，也是所有传智人共同努力的目标。为此，十三年来，传智人不曾有一丝懈怠，相信在传智人的不懈努力下，大道不远，终在脚下。

我们的服务理念：用爱成就每一位学生

爱是教育的灵魂，教育根植于爱。作为教育从业者要发自内心地关爱学生，全心全意地培养学生，竭尽所能成就学生的梦想。
